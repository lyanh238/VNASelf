User Profile Visibility Test

Test Cases

Test 1: Check if user profile is visible in expanded sidebar
Test 2: Check if user profile is visible in collapsed sidebar
Test 3: Check if user profile stays at bottom of sidebar
Test 4: Check if user profile is clickable

Expected Behavior

  • User profile should always be visible at the bottom of sidebar
  • User profile should show user avatar and name when expanded
  • User profile should show only avatar when collapsed
  • User profile should be clickable and show dropdown

Steps to Test

  1. Open the application
  2. Check if user profile is visible at bottom of sidebar
  3. Click on user profile - should show user info
  4. Collapse the sidebar
  5. Check if user profile is still visible (should show only avatar)
  6. Expand the sidebar again
  7. Check if user profile is still visible with full info

Fixed: Added min-height and max-height to ensure consistent sizing
